A collection of specialized AI agent skills and configurations designed to improve communication, automate tasks, and enforce best practices across projects.

## Included Skills

- **`caveman`**: Ultra-compressed communication mode to minimize token usage without losing technical accuracy.
- **`caveman-commit`**: Generates concise, "why-focused" Conventional Commits.
- **`caveman-review`**: Produces ultra-compressed, actionable code review comments.
- **`create-readme`**: Intelligent README file generator for open-source projects.
- **`gsap`**: Comprehensive GSAP animation reference for complex UI interactions.
- **`security-best-practices`**: Performs language and framework-specific security audits.
- **`ui-ux-pro-max`**: Extensive UI/UX design intelligence for web and mobile platforms.
- **`vercel-react-best-practices`**: Performance optimization guidelines from Vercel Engineering.

## Core Structure

- `.agents/skills`: Contains the installed skill definitions and their respective `SKILL.md` files.
- `AGENTS.md`: Agent security policies and execution constraints.
- `skills-lock.json`: Dependency lockfile for the installed agent skills.
