# Project Management

Skills for planning, grilling ideas, and turning conversation into work artifacts.

- `grill-me`: exhaustive design interview for plans and decisions.
- `grill-with-docs`: same grilling flow, but grounded in project docs and ADRs.
- `to-prd`: converts discussion into a PRD.
- `to-issues`: breaks work into vertical-slice issues.
- `write-a-skill`: scaffolds new skills with the expected file structure.
